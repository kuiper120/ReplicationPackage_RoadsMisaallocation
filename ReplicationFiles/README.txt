README-Replication Files for “Rural Roads and Factor Misallocation in Agriculture”, April 2026

This folder contains the replication files for the manuscript “Rural Roads and Factor Misallocation in Agriculture.”

Overview: The analysis uses household-, plot-, and enumeration-area-level data constructed from the Ethiopia Socioeconomic Survey and road-surface information from the community questionnaire. The empirical analysis estimates difference-in-differences specifications comparing enumeration areas upgraded earlier to those upgraded later, restricting the sample to eventually improved areas.

Folder contents

1. do_tables.do
   Main Stata do-file used to generate the tables reported in the manuscript.

2. misalloc_household.dta
   Household-level analysis dataset used for the main misallocation and efficiency-gap regressions.

3. misalloc_plotlevel.dta
   Plot-level dataset used for within-household misallocation analysis.

4. misalloc_village.dta
   Enumeration-area-level dataset used for village-level misallocation and dispersion measures.

5. misalloc_within_hh.dta
   Dataset used to examine within-household allocation measures.

6. ea_road_surface_treatment.dta
   Enumeration-area road treatment dataset, including treatment timing.

7. sum_productivity.dta
   Dataset contain productivity measures used to perform covariate balance check.

8. tables/
   Output folder containing the tables generated by the Stata do-file.

Software requirements

The replication files are written for Stata V.17. The code uses standard Stata commands together with commonly used user-written packages for high-dimensional fixed effects and table export. Users should install any missing packages before running the replication script.

Instructions

1. Open do_tables.do in Stata.
2. Update the working directory at the top of the file to the location of the replication folder.
3. Run the do-file from beginning to end.
4. The generated tables will be saved in the tables/ subfolder.


Notes

The replication package contains the cleaned analysis datasets required to reproduce the tables in the manuscript. Monetary values are expressed in constant U.S. dollars. Standard errors are clustered at the enumeration-area level unless otherwise noted in the tables.
